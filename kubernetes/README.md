# Kubernetes

This directory contains Kubernetes labs, manifests, and deployment exercises focused on container orchestration.

## Topics Covered

- Pods
- ReplicaSets
- Deployments
- Services
- ConfigMaps
- Secrets
- Persistent Volumes
- Ingress
- Namespaces
- Helm
- Troubleshooting

## Goal

Gain hands-on experience deploying, managing, and troubleshooting applications on Kubernetes while following cloud-native best practices.
